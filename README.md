# Bootstrap-FontAwesome-webpack-app

Empty project.

## Building and running on localhost

First install dependencies:

```sh
npm install
```

To create a production build:

```sh
npm run build-prod
```

To create a development build:

```sh
npm run build-dev
```

## Run dev server

```sh
npm run dev-server
```

## clear dist folder

```sh
npm run clear
```
